This project is coded entirely in java usign the java.swing library, Some of the concepts included in this project include inheritance, CSV files, polymorphism, 2D arrays and try-catch statements.

This is an app for teachers to be able to keep track of their student's grades. Firstly, there is a login page which allows teachers and administrators to sign in, Teachers are able to create classes and add grades for each student in the class for the various exams taken throughout the semester. Additionally, the teachers can create line charts, bar charts and pie charts of the student's grades. Fianlly, administrators are able to add and remove students from the school database.
